# Hand Gesture Sudoku Solver

An interactive Sudoku game controlled entirely using hand gestures through a webcam.

This project uses **MediaPipe** for real-time hand tracking and **OpenCV** for rendering an interactive Sudoku interface, allowing users to select cells and input numbers using their index finger.

## 🚀 Features

* Real-time hand tracking
* Gesture-based interaction (point & hold)
* Interactive Sudoku grid
* Dynamic number selection panel
* Touchless gameplay experience

## 🛠 Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy
